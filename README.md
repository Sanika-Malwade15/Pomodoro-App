# 📖 About the Project

**ZenPulse** is a modern productivity web application designed to help users maintain focus, manage tasks, and improve work efficiency using the **Pomodoro Technique**.

It combines a customizable Pomodoro timer, integrated task management, calming ambient soundscapes, and productivity analytics into a clean and intuitive interface. Whether you're studying, coding, or working on personal projects, ZenPulse provides everything needed to stay organized and focused.

---

#  Features

###  Pomodoro Focus Timer
- Focus, Short Break, and Long Break modes
- Play, Pause, Reset, and Skip controls
- Full-screen focus mode
- Custom session durations

###  Smart Task Manager
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Display active task during focus sessions

###  Ambient Soundscapes
-  Rain
-  Ocean Waves
-  Forest
-  Alpha Beats
- Adjustable volume controls
- Continuous background playback

###  Productivity Analytics
- Track completed Pomodoro sessions
- Monitor total focus time
- Real-time productivity statistics

###  Theme Support
- Light Mode
- Dark Mode

###  Custom Settings
- Configure Focus duration
- Configure Short Break duration
- Configure Long Break duration

# 🚀 Getting Started

## Prerequisites

Make sure you have installed:

- Node.js (v14 or above)
- npm

Check installation:

```bash
node -v
npm -v
```

---

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ZenPulse.git
```

Move into the project folder:

```bash
cd ZenPulse
```

Install dependencies:

```bash
npm install
```

Install Lucide React:

```bash
npm install lucide-react
```

---

## ▶️ Run the Project

For Vite:

```bash
npm run dev
```

For Create React App:

```bash
npm start
```
